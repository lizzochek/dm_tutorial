1.9 Алгебраїчні структури
Алгебраїчною структурою називається структура з певною
множиною M, невеликою кількістю бінарних алгебраїчних операцій та
такими аксіомами, як асоціативність, комутативність, дистрибутивність.
У алгебраїчних структурах розрізняють операції типу множення «⋆»
та типу додавання «+». Найчастіше, ці операції не співпадають з
операціями арифметичного множення та додавання, хоча операцію «⋆»
називають добутком, а «+» — сумою.
Для операцій типу множення завжди знайдеться нейтральний
елемент e, множення на який додавання якого залишає результат незмін-
ним, тобто, ∃ е ∈ M(x ∈ M; x ⋆ e = x). Аналогічний елемент є для операції
додавання, ∃ е ∈ M(x ∈ M; x + e = x).
Структури з такими операціями називають мультиплікативними та адитивними, відповідно.
Комутативна структура — це така, яка має аксіому комутативності, тобто
∀  a, b ∈ M (a ⋆ b = b ⋆ a).
Асоціативна структура — це така, яка має аксіому асоціативності,
тобто
∀  a, b, с ∈ M ( (a ⋆ b) ⋆ с = a ⋆(b ⋆ с) ) .
Моноїд – це асоціативна структура, яка має одну операцію типу
множення та нейтральний елемент.
Моноїдами можуть бути множина натуральних чисел з операціями
додавання чи множення. При додаванні нейтральним елементом є нуль, а
при множенні — одиниця. Важливою властивістю моноїда, як і будь-якої
структури є та, що результат операції з будь-якими елементами повинен
належати до тієї самої множини. Наприклад, структура парних чисел з
операцією «+» є моноїдом, а непарних чисел — ні.
Незвичним моноїдом є множина усіх слів з алфавітом, наприклад,
{a, b, c} з операцією конкатенації «○». Для нього справджується асоціатив-
ність, як наприклад,
(aba ○ baba) ○ caca = aba ○(baba ○ caca) = abababacaca.
Група — це алгебраїчна структура G = 〈G; ⋆〉 з наступними аксіомами:
Г1 : ∀ a,b,с ∈ G ( (a ⋆ b) ⋆ с = a ⋆ (b ⋆ с) ),
тобто, групова операція асоціативна.
Г2 : ∃ е ∈ G (x ∈ G; x ⋆ e = e ⋆ x = x), у групі є одиничний елемент.
Г3 : ∀ а ∈ G ∃ а –1 ∈ G (а ⋆ а –1 = а –1 ⋆ а = е), тобто, для кожного елемента а
групи знайдеться йому обернений елемент а –1 .
Абелева або комутативна група — це група, яка має додатково
аксіому комутативності.
Скінченна група — це група, множина якої G є скінченною.
Відповідно, якщо множина G є нескінченною, то і група є нескінченною.
Число елементів скінченної групи є її порядком.
Прикладом групи є множина раціональних чисел з операцією
множення. У ній нейтральний елемент е = 1. Для неї для дробу виду a/b
завжди знайдеться обернений елемент b/a. І це є абелева група, бо
a/b * c/d = c/d * a/b
Ця група є нескінченною, бо повинен знайтись результат операції
для будь-яких великих чисел.
Щоб множина чисел G була скінченною, часто використовують
операцію взяття по модулю порядку групи р = |G|. Наприклад, якщо р = 10,
21то результат добутку елементів 7 та 7 за модулем р буде 7 ⋆ 7 = 9. Так
створюються, так звані, групи модульної арифметики.
Наприклад, 8-бітові числа у мікропроцесорах та операції додавання
чи множення формують групи арифметики за модулем 2 8 = 256.
Кільце — це алгебраїчна структура, у якій визначені дві асоціативні
операції — додавання і множення, причому за додаванням кільце є
комутативною групою, а за множенням — воно дистрибутивне відносно
додавання. Операцію множення, як правило, позначають точкою. Тобто,
кільце — це структура
K = 〈M; *, +〉,
для якої виконуються аксіоми:
К1 : ∀ a, b, с ∈ M ( (a + b) + c = a + (b+ c) ).
К2 : ∀ a, b ∈ M ( a + b = b+ a ).
К3 : ∃ 0 ∈ M ∀ a ∈ M ( a + 0 = a ).
К4 : ∃ 0 ∈ M ∀ a ∈ M ( a + (–a) = 0 ).
К5 : ∀ a, b, с ∈ M ( (a * b) * c = a * (b * c) ).
К6 : ∀ a, b, с ∈ M ( (a + b)* c = a * c + b * c; a * (b + c) = a * b + a * c ).
Комутативне кільце — це кільце, у якого є ще аксіома
комутативності множення:
К7 : ∀ a, b ∈ M ( a * b = b * a ).
Унітарне комутативне кільце (кільце з одиницею) — це
комутативне кільце, у якого є ще аксіома множення на одиницю:
К8 : ∃1  ∈ M ∀ a ∈ M ( a * 1 = 1 * a = а ).
Основним прикладом унітарного комутативного кільця є кільце
цілих чисел.
На жаль, кільце не можна вважати мультиплікативною групою, бо
для елемента 0 немає йому оберненого елемента. Щоби перетворити
кільце у мультиплікативну групу, слід додати ще одну аксіому або
теорему:
Т8 : ∀ a ∈ M, а ∈ 0 ∃ а –1 ∈ M ( a * а –1 = 1 ).
Комутативне поле — це комутативне кільце, для якого також
виконується аксіома Т 8 .
Отже, у поля є дві операції: додавання та множення, причому усі
елементи поля за додаванням і ненульові елементи за множенням форму-
ють комутативні групи, а операції додавання і множення пов’язані між
собою дистрибутивним законом.
Типовими прикладами комутативних полів є поле раціональних
чисел і поле реальних чисел. Серед скінченних комутативних полів
важливе місце займають поля Галуа.
