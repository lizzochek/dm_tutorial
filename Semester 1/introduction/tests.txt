Питання та завдання
1. Виконайте вручну алгоритм Евкліда для a = 231 та b = 182.
2. Виконайте вручну алгоритм розрахунку 1+3+5+...+(2п – 1) = п 2 для
п = 16.
3. Розв’яжіть діофантове рівняння x 2 + y 2 = 100.
4. Розв’яжіть діофантове рівняння x 2 – y 2 = 4.
5. Знайти sqrt(a) за алгоритмом Герона: х1 = 1; х n+1 = 0.5(xn + a/xn) = sqrt(a)
n = 1,2,,, для a = 256  з використанням раціональних чисел.
6. Знайти sqrt(a) за давньоіндійським алгоритмом: х1 = sqrt(a) (перше наближення); 
qn = (a - xn^2)/2xn
xn+1 = xn + qn - qn^2/2(xn+qn) = sqrt(a)
n = 1, 2 для а = 16384 з використанням раціональних чисел.
7. Довести графічно теорему Піфагора.
8. Розрізати тупокутний трикутник на кілька гострокутних трикутників.
